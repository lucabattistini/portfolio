# lucabattistini.dev

Personal portfolio — hands-on software engineering with experience in technical leadership roles.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **State:** XState v5 + @xstate/react
- **3D:** React Three Fiber + drei (interactive particle portrait)
- **Animation:** Motion (Framer Motion v12)
- **Language:** TypeScript
- **Deploy:** Vercel via GitHub Actions

## Development

```bash
pnpm install
pnpm dev
```

## Commands

| Command       | Description                     |
| ------------- | ------------------------------- |
| `pnpm dev`    | Start dev server (Turbopack)    |
| `pnpm build`  | Production build                |
| `pnpm lint`   | ESLint + TypeScript type-check  |
| `pnpm format` | Prettier (with Tailwind plugin) |
